Key Features
============

INPUT
-----
- List of LinkedIn profile URLs (3-10 recommended)
- Supports file input or command-line URLs
- Validates LinkedIn URLs automatically

PROCESSING
----------
- Scrapes public LinkedIn data (no API key needed)
- Extracts: name, headline, experience, education, skills, posts
- Uses AI to synthesize composite persona from patterns
- Supports multiple AI models (Qwen/free, GPT-4, Claude, Gemini)

OUTPUT
------
- Markdown persona document (12 sections)
- JSON profile data (raw scraped data)
- Actionable content testing framework

UNIQUE FEATURES
---------------
1. Content Testing Framework - Score content on relevance, engagement, action
2. Example Headlines - Specific content that would resonate
3. Anti-Patterns - What to avoid
4. Communication Preferences - Tone, format, detail level
5. Free Option - Qwen model requires no API key

Usage
=====

INSTALL:
  cd ~/Development/Scripts/PersonaGenerator
  ./install.sh

TEST:
  ./test_persona.sh

GENERATE:
  python3 persona_generator.py --urls sample_urls.txt

VIEW:
  cat output/composite_persona.md
